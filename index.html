SIGED — Sistema Integrado de Gestão Educacional

Plataforma web completa desenvolvida do zero para a Secretaria Municipal de Educação, centralizando toda a gestão educacional do município em um único sistema. Inclui gestão de funcionários com ponto eletrônico e controle de cargos por roles de acesso; gestão de escolas, merenda, notas, faltas, registros e planos de aula; emissão automatizada de documentos oficiais; gestão do transporte escolar intermunicipal; e módulo de atendimento psicológico com agendamento. Conta com portal do aluno para consulta de notas, faltas, provas e calendário escolar; aplicação de provas online; auto cadastro de funcionários com aprovação administrativa; e sistema de notificações inteligentes para alertar sobre alunos com baixo desempenho, duplicidades, vínculos de turma pendentes e novos cadastros. Hospedado em VPS própria.

Node.js JavaScript PostgreSQL Prisma ORM VPS REST API
